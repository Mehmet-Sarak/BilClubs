Desktop App

Required to access BilClubs

BilClubs is the official club and events platform for Bilkent University students. It brings together all campus clubs in one place, letting students discover communities that match their interests, follow upcoming events, and manage their memberships.

The platform is built around a desktop application — there is no web interface. What you are seeing now is the BilClubs API server, which the desktop app communicates with in the background. It is not designed to be accessed from a browser, and no content is available here.

If you are a student looking to explore clubs and events, please download and install the BilClubs desktop app. If you are a developer and need access to API documentation, get in touch with our team.

Questions? Contact our support team and we'll get you sorted.
